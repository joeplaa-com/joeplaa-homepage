### Hosting
1. Static websites will be hosted through [AWS CloudFront](https://aws.amazon.com/cloudfront/).
2. Dynamic websites will be hosted on [AWS LightSail](https://aws.amazon.com/lightsail/).
3. Option three is currently only possible if you host the site yourself.
